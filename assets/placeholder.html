ASH Foundation Website - Image Assets

Required Images

To complete the website, please download the images from the Figma design and place them in this assets/ folder.

Steps to Download Images from Figma:

  1. Open the Figma design link
  2. Click on each image layer
  3. Use the Export panel on the right side
  4. Select PNG or SVG format as appropriate
  5. Click "Export" and save to this folder

Image List:

  • logo.png - ASH Foundation logo
  • flag-bd.png - Bangladesh flag
  • hero-image.jpg - Main hero section image
  • about-image.jpg - About section image
  • fund-*.jpg - 6 donation fund images
  • project-*.jpg - 3 project images
  • blog-*.jpg - 3 blog post images
  • event-*.jpg - 3 event images
  • video-*.jpg - 3 video thumbnail images
  • gallery-*.jpg - 8 gallery images
  • world-map.jpg - World map image
  • *.svg - Social media icons
Note: The website will display placeholder text where images are missing. Once you add the actual images from Figma, the website will look exactly as designed.
