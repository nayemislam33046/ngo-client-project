# Alhaj Shamsul Haque Foundation Website

A modern, responsive website for the Alhaj Shamsul Haque Foundation (ASHF), a registered non-governmental voluntary organization.

## Project Structure

```
.
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript functionality
├── assets/             # Images and media files
└── README.md           # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling and active states
- **Donation Form**: Interactive donation submission with validation
- **Video Gallery**: Playable video thumbnails
- **Image Gallery**: Clickable images with modal view
- **Live Chat**: Floating chat button
- **Global Presence Map**: Interactive map showing office locations
- **Modern UI**: Clean design based on Figma specifications

## How to Download Images from Figma

1. Open the Figma design: https://www.figma.com/design/Bd7PQoqma2RmpzLNrejTXQ/Alhaj-Shamsul-Haque-Foundation?node-id=220-767
2. Select each image layer
3. Right-click and choose "Export" or use the Export panel on the right
4. Choose the appropriate format (PNG for photos, SVG for icons)
5. Save to the `assets/` folder with the names listed above

## Installation

1. Clone or download this repository
2. Add all required images to the `assets/` folder
3. Open `index.html` in a web browser
